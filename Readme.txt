Act Stop Spam is an automated wp plugin to check and block spam IP Address.

== Description ==

Protect your site from spammers by using Act Stop Spam plugin.Act Stop Spam is an automated script to check and block Spam IP Address based on StopForumSpam.com blacklisted ip database. Checks only the IP address and takes advantage of Stop Forum Spam's "confidence" assessment.

Act Stop Spam Plugin provides spam protection.

*	Simple protection from spammers.
*	Checks spammer ip records from stopforumspam.com database and protects your site.
*	Saves your site and time from incoming spam bots.
*	Prevents excessive usage bandwidth from spam bots.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `act-stop-spam` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can customize default settings  through 'Settings>Act Stop Spam' in WordPress Settings menu.
4. You can monitor the list of blocked spam ips through 'Act Stop Spam' menu in Wordpress.

== Frequently Asked Questions ==

= How Act Stop Spam works? =

Activating 'Act Stop Spam' plugin enables protection for your site from spammers.It verifies Guest IP address with stopforumspam.com according to the confidential level.If IP is listed in spam database ,access to site from that IP will be blocked with Custom message provided in 'Act Stop Spam' settings page.

= Does this require special component in server ? =

This plugin required file_get_contents inbuilt php function or PHP Curl which will be installed default in server.You can contact your server administrator if it is blocked.
